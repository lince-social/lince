# Objective

Automate the creation of the This Month in Lince presentation.
The main.typ is used for reading the latest month inside this directory and seeing it's data.json.
The data.json has three languages, Brazillian Portuguese, Mandarin and English. The text will be written in one of them, you need to fill out the other two.
Your job is looking at the Slides in this file and for each "-" line create a slide with the exact text, if there is an image referenced at the start of the line put it in the json.
